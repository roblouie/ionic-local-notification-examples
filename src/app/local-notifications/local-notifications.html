Local Notifications

Media and Data

All buttons will trigger a notification one minute from when you press them. This is to give you enough time to press the home button on iOS 10, since local notifications will not pop if the app is in the foreground.

Triggers a local notification with no icons set. If you have ic_popup_reminder.png in all your res/drawable-* folders that will be used. Otherwise you will get the default bell icon.

Sets the icon to be a specific file in a specific folder in the res folder. Since small icons are required for notifications, if you only set a single icon it will be treated as the small icon. This means it will also be turned into a single color icon with transparency.

Here we set both the large and the small icons. The small icon as always is turned to a single color icon with transparency, but the large icon will remain full color. You can only have a large icon if you also have a small one.

Small and large icons again, but this time using an image in the assets folder for the large icon instead of using a resource. You can only do this for the large icon, small icon is res only.

Plays a custom sound effect when the notification triggers.
